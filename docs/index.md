# Forge Documentation

Forge is an AI-powered development platform that helps developers build software with intelligent AI agents.

> **🚀 Beta Launch Notice**: Several advanced UI features are temporarily disabled for the beta launch to provide a cleaner, more focused user experience. All components and functionality remain intact and can be easily re-enabled post-beta. See the [Beta Launch section in Advanced Features](advanced_features.md#beta-launch-temporarily-disabled-ui-features) for complete details and re-activation instructions.
