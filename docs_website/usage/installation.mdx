## Forge Cloud

The easiest way to get started with Forge is on Forge Cloud, which comes with $20 in free credits for new users.

To get started with Forge Cloud, visit [app.all-hands.dev](https://app.all-hands.dev).

For more information see [getting started with Forge Cloud.](/usage/cloud/Forge-cloud)

## Running Forge on Your Own

Run Forge on your local system and bring your own LLM and API key.

For more information see [running Forge on your own.](/usage/local-setup)
