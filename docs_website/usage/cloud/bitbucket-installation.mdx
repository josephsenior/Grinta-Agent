## Prerequisites

- Signed in to [Forge Cloud](https://app.all-hands.dev) with [a Bitbucket account](/usage/cloud/Forge-cloud).

## IP Whitelisting

If your Bitbucket Cloud instance has IP restrictions, you'll need to whitelist the following IP addresses to allow Forge to access your repositories:

## Adding Bitbucket Repository Access

Upon signing into Forge Cloud with a Bitbucket account, Forge will have access to your repositories.

## Working With Bitbucket Repos in Forge Cloud

After signing in with a Bitbucket account, use the `select a repo` and `select a branch` dropdowns to select the
appropriate repository and branch you'd like Forge to work on. Then click on `Launch` to start the conversation!
