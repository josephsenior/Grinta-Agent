## Obtain Your Forge LLM API Key

1. [Log in to Forge Cloud](/usage/cloud/Forge-cloud).
2. Go to the Settings page and navigate to the `API Keys` tab.
3. Copy your `LLM API Key`.

![Forge LLM API Key](/static/img/Forge-llm-api-key.png)

## Configuration

When running Forge, you'll need to set the following in the Forge UI through the Settings under the `LLM` tab:
- `LLM Provider` to `Forge`
- `LLM Model` to the model you will be using (e.g. claude-sonnet-4-20250514)
- `API Key` to your Forge LLM API key copied from above

## Using Forge LLM Provider in the CLI

1. [Run Forge CLI](/usage/how-to/cli-mode).
2. To select Forge as the LLM provider:
  - If this is your first time running the CLI, choose `Forge` and then select the model that you would like to use.
  - If you have previously run the CLI, run the `/settings` command and select to modify the `Basic` settings. Then
    choose `Forge` and finally the model.

![Forge Provider in CLI](/static/img/Forge-provider-cli.png)

## Pricing

Pricing follows official API provider rates. [You can view model prices here.](https://github.com/BerriAI/litellm/blob/main/model_prices_and_context_window.json)

For `qwen3-coder-480b`, we charge the cheapest FP8 rate available on openrouter: \$0.4 per million input tokens and \$1.6 per million output tokens.
