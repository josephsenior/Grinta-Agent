### Chat Panel
- Displays the conversation between the user and Forge.
- Forge explains its actions in this panel.

### Changes
- Shows the file changes performed by Forge.

### VS Code
- Embedded VS Code for browsing and modifying files.
- Can also be used to upload and download files.

### Terminal
- A space for Forge and users to run terminal commands.

### Jupyter
- Shows all Python commands that were executed by Forge.
- Particularly handy when using Forge to perform data visualization tasks.

### App
- Displays the web server when Forge runs an application.
- Users can interact with the running application.

### Browser
- Used by Forge to browse websites.
- The browser is non-interactive.
